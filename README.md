# QWELプログラミング教室 (SPA)

SPA = シングルページアプリケーションとして動作する、プログラミング教室のwebページ。  
背景で動き続けるアニメーションと、ページ遷移が特徴的。  

[https://ac.qwel.design/](https://ac.qwel.design/)

---

## 変遷

- 2018年 東京都町田市金森にて個人のプログラミング教室を開き、運用開始  
- 2023年 福井県福井市越前海岸エリアへ移住したため、コンテンツをリメイク、またソースコードをリファクターした上、再運用開始  
- 2026年 1月現在プログラミング教室は、QLEP(QWEL Local Engine Project) の形態で稼働中
  - [QLEP Notes](https://github.com/qweldesign/qwel-local-engine-notes)
  - [QLEP Devlog](https://github.com/qweldesign/qwel-local-engine-devlog)
- ただし、生徒募集は打ち切ったため、webページは運用終了

## 内容

主にプログラミング教室で大切にしたいことについて書かれており、  
QWELが「星の王子さま」の愛読家であることであったり、  
前職がピアノ調律師であったことであったり、  
「教えてgoo!（知恵袋）」の回答者として、コンピューター・テクノロジーの分野で、  
1ヶ月間に25個のベストアンサーを獲得したことであったり、  
読んでみると、その他意外な一面が見つかるかも知れない―。
